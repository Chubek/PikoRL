Chapter 7 - Documentation Pipeline
==================================

Documentation source sets:

- API headers and C sources;
- ``docs/FrontPage.md``;
- ``docs/manual/*.md``.

Pipeline behavior:

- Doxygen parses API docstrings and manual pages in one configured run;
- ``FrontPage.md`` is the main page;
- HTML/LaTeX/XML/DocBook outputs are generated by configured options.

Invocation:

- configure with ``PICORL_BUILD_DOCS=ON``;
- build target ``docs``.
